1. 분할 압축된 파일 2개를 압축해제
2. Dolphin 2021.3.1이상 버전의 안드로이드 스튜디오를 실행
3. Projects - Open - '프로젝트 폴더의 경로' - OK 클릭

## 빌드와 앱 실행
